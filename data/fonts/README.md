# Fonts

Place the following TTF font files here for the best terminal rendering experience.
The application falls back to Dear ImGui's built-in font if these are not found.

## Recommended (monospace — terminal feed text)

**JetBrains Mono** (free, SIL Open Font License)
Download: https://www.jetbrains.com/lp/mono/
File needed: `JetBrainsMono-Regular.ttf`

**IBM Plex Mono** (free, SIL Open Font License)
Download: https://github.com/IBM/plex/releases
File needed: `IBMPlexMono-Regular.ttf`

## Recommended (condensed — headers)

**IBM Plex Sans Condensed** (free)
Download: https://github.com/IBM/plex/releases
File needed: `IBMPlexSansCondensed-Regular.ttf`

**Inter** (free)
Download: https://rsms.me/inter/
File needed: `Inter-Regular.ttf`
